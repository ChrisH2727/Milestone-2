How To Get Started
1. Enter a short task description. task start date and a best case, most likely case and worst case task duration into the Task Table below.
2. Enter as many tasks as you need to. All tasks have an automatic finish to start relationship.
3. Select the Non Project Days allocted to your team. Note that Saturday & Sunday are set by default as Non Working Days.
4. When ready click the Start button.
5. You can add more tasks, change task durations and change Non Project Days at any time and the task table will automatically update, but you must click the Start button to run the simulation.
6. Click the Reset button to clear the Task Entry Table.
